backendjava 工程

mybatis-generator

2020-04-21
analyst, java

步骤

  1. 创建 maven 项目，如 generator-mapper。
  2. 配置 pom.xml。
  3. 配置 resources/generatorConfig.xml。
  4. maven 面板点击 Plugins - mybatis-generator:genrate。

QA

  • Communications link failure 报错：数据库端口配置错误等，参看 “Communications link failure”错误
  • Unknown system variable ‘query_cache_size’ 报错：mysql 驱动程序版本与 mysql 服务器版本不一致，查看 Unknown system variable ‘query_cache_size’
