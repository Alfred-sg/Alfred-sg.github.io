spring boot试水

发表于 2019-03-07 | 分类于 spring boot |

idea 搭建项目

  1. 安装 java, maven, idea
  2. create project - spring initializr - 设置 name, GroupId, ArtifactId 等 - 设置 dependencies，暂时选择 web - 设置 project name
  3. 删除 .mvn, mvnw, mvnw.cmd
  4. 加载依赖、插件，Run/Debug Configuration 会自动生成
  5. localhost:8080 访问页面
  6. 编写 controller

配置文件

将 resource 中的 application.properties 改为 application.yml，spring boot 支持 yml 配置。

配置文件中的内容可通过 @Value(‘${ymlVariablePath}’) 注解将配置数据注入 bean 属性或方法的参数中，或者使用 @ConfigurationProperties(prefix = ymlVariablePrefix) 注解构建 POJO 读取。其中，ymlVariablePath 为 yml 文件中以 “.” 分割的配置项属性名路径，ymlVariablePrefix 为配置项属性名前缀。如配置项层级较深，在构建 POJO 时需要单独对深度嵌套的数据封装实体类。如 bean 中的静态属性需要获取配置项数据，可通过 @Value 构造 set 方法实现。

针对不同的环境，可以创建 application-dev.yml, application-test.yml, application-prod.yml 等文件，并在 application.yml 指定当前所采用的配置文件。

说明

@RestController：指定 controller 以 rest 风格返回 json 数据
@RequestMapping：指定路由、请求方法
